# Invisible on screen share

OnCue's overlay uses Windows **display-affinity** controls, so it's **excluded from screen capture** — Zoom, Google Meet, Microsoft Teams, OBS, and the OS recorder all skip it. The other side sees only the window or screen you chose to share, never the overlay on top of it.

## Hide instantly

A configurable hotkey hides the overlay immediately. Set it in **Settings → Shortcuts**.

## Responsible use

OnCue is a tool for thinking on your feet. Use it in line with the policies of your calls, employer, and local laws — e.g. around recording or assistance during interviews and exams.

## Privacy

Audio is transcribed to power live advice. For what's stored and how data is handled, see the [privacy policy](https://oncue.fun/privacy).
