# Getting started

## 1. Install

Download the Windows installer from [oncue.fun/download](https://oncue.fun/download) and run it. Supports **Windows 10 and 11 (64-bit)**. After the first install, OnCue **auto-updates** in the background.

> Windows may show a SmartScreen prompt on first launch. Choose **More info → Run anyway**.

## 2. Sign in

Sign in with your email. You'll get a one-time code — no password to remember.

## 3. First session

1. Click **Start Session** on the overlay.
2. Name it and optionally add a guideline — e.g. *"Discovery call, stay consultative."*
3. Pick input sources (your voice + system audio are on by default).
4. Start talking. Transcription appears live; the advisor starts suggesting cues.

**End Session** generates a summary and action items.

## The overlay

* Floats on top of other windows and is **invisible to screen share**.
* A hide hotkey dismisses it instantly (configurable in **Settings → Shortcuts**).
* Scroll, or Shift+scroll, to move through advice and reply cards.
