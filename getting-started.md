# Getting started

## 1. Install

Download the Windows installer from [oncue.fun/download](https://oncue.fun/download) and run
it. OnCue supports **Windows 10 and 11 (64-bit)**. After the first install, the app
**auto-updates** in the background — no manual reinstalls.

> Windows may show a SmartScreen prompt on first launch. Choose **More info → Run anyway**.

## 2. Sign in

Launch OnCue and sign in with your email. You'll get a one-time code — enter it to
authenticate. No password to remember.

## 3. Run your first session

1. Click **Start Session** on the overlay.
2. Give it a name and (optionally) a guideline — e.g. *"Discovery call, stay consultative."*
3. Pick your input sources (your voice + system audio are on by default).
4. Start talking. Transcription appears live, and the advisor begins suggesting cues.

When you're done, **End Session** generates a summary and action items.

## The overlay

* It floats on top of your other windows and is **invisible to screen share**.
* Press your hide hotkey to dismiss it instantly (configurable in **Settings → Shortcuts**).
* Scroll, or use Shift+scroll, to move through advice and reply cards.
